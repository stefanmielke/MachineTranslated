# １４７話　興味深い

「アイン……うん、覚えた。了解。よろしくね」

「よろしく頼むよ」

<b>

シルファとアインは握手を交わした。

<b>

「む？」

「どうしたんだい？」

「ううん、なんでもないよ」

<b>

握手をしたことで、シルファは、アインの力量をなんとなくではあるが把握した。

それなりに強い。

しかし、あくまでもそれなり、というレベル。

<b>

自分やハルには遠く及ばない。

なにかよからぬことを企んでいたとしても、力づくでどうこう、ということはできないだろう。

<b>

シルファは混乱した。

<b>

相当な実力者だと思っていたのだけど……

でも、そんなことはない。

普通の一般人なのだろうか？

<b>

いや、そんなことはない。

<b>

シルファは己の感覚を信じることにした。

最初に感じた危機感は、確かに本物だ。

きっと、アインが巧妙に隠して、一般人になりきっているに違いない。

そう判断した。

<b>

「それで、どこに案内すればいいのかな？」

「倉庫。知っている？」

「ああ、それなら近くにあるよ。おいで、こっちだよ」

<b>

アインが先を歩いて、シルファが後ろに続く。

<b>

「ところで」

<b>

なにげない様子でアインが尋ねる。

<b>

「キミは、どうして怖い顔をしているのかな？」

「……そんな顔、しているかな？」

「見た目はよくわからないんだけどね。でも、とても怖い顔をしているよ。まるで、僕を敵と判断して、警戒しているかのようだ」

「驚いた。洞察力に優れているんだね」

「まあ、色々とあってね。それで、どうしてなのかな？」

<b>

シルファは迷う。

正直なところを話したら、この男は攻撃的な態度にならないだろうか？

<b>

それはないか、と考え直した。

<b>

危機感を覚えたことは確かではあるが、敵意は感じられない。

そもそも、害を与えようとしているのなら、すでに行動に移っていてもおかしくはない。

のんびりとおしゃべりをする必要がどこにある？

<b>

「アインが、とても強い人に感じられた。シルファが危機感を覚えるほど」

「僕が？」

「そう。とんでもない修羅場を潜り抜けてきた、死地に何度も飛び込んできて……そんな人に見えた。そういう人だけが持つ、鋭い刃のような気配がした」

「ああ、なるほど。そういうことか。それは、僕のことじゃないよ」

「？」

「詳しく教えることはできないんだけど、僕は、これでもそれなりの身分なんだ。だから、いつも護衛を従えている」

「見えないけど？」

「密かに見守っている、というヤツさ。いつでもどこでも、なにが起きても瞬時に対処できる、とても優秀な護衛さ。きっと、彼の気配を感じ取り、僕のものと勘違いしたんだろう」

「なるほど……言われてみれば、気配が少し違う。納得」

<b>

答えが明かされてみれば、なんてことのない話だった。

東方の国の忍と呼ばれる者のように、護衛はアインの傍に控えて……しかし、姿を見せることなく、影から守っているのだろう。

よくある話だ。

<b>

……と、シルファは納得するものの、よくある話などではない。

まったく気配を感じさせず、姿も見せない護衛がいるなんて、なかなかある話ではない。

少し前まで闇に生きてきたシルファだからこそ理解できる話であり、一般人は首を傾げるだろう。

<b>

自分の常識がズレていることに気がつかないシルファは、アインの言動におかしなところはないと判断して、さらに警戒を解いた。

事実、アインはなにをするわけでもなく、シルファを倉庫に案内する。

<b>

「倉庫はここだよ」

「ありがとう。なにかお礼をしたい。してほしいことはある？」

「気にしないでいいよ」

「借りは返さないと。本当に、なにもない？」

「うーん……それじゃあ、今度は僕の用事を手伝ってくれるかな？」

「どんな用事？」

<b>

シルファは倉庫に荷物を置いて、それから問いかけた。

<b>

「人を探しているんだ。たぶん、学院にいると思うんだけど……でも、ここはとても広いだろう？　だから、なかなか見つけることができなくて」

「なるほど。なら、シルファも手伝うよ。どんな人？」

「クラウディア・ファナシス、っていう女の子なんだけど、知っているかな？」

「おぉ。金髪ツインテールドリル？」

<b>

シルファは、学院に来た時に一度、顔を合わせただけなのだけど、しっかりと覚えていた。

これでも記憶力は良い方なのだ。

<b>

「ぷっ」

「どうしたの？」

「いや……まさか、あの子をツインテドリルなんて呼ぶ子がいるなんて……あははっ、ダメだ。ツボに入った。あはははっ」

「？」

<b>

シルファは小首を傾げた。

笑えるようなことはなにも言っていないはずなのに、なぜ？

<b>

「ふぅ……なかなかどうして、たくさん笑わせてもらったよ。こんなに楽しい気分は、本当に久しぶりだ。ありがとう」

「どういたしまして？」

「でも、クラウディアを知っているのなら、話は早いね。顔がわかるのなら、説明する必要はないし、それなりに早く見つかりそうだ」

「うん、がんばるよ」

「期待しているよ」

<b>

二人は倉庫を出て、学内を歩く。

特に目的地は定めていない。

まずは適当に歩いて、聞き込みをして、それで捜索範囲を絞っていこう、という方針なのだ。

<b>

「そういえば」

<b>

思い出した様子で、シルファが問いかける。

<b>

「どうして、クラウディアを探しているの？」

「んー……詳細を話すことはできないのだけど、一言で言うのなら、家族会議をするためかな」

「家族会議？」

「クラウディアは、僕の妹なのさ。アイン・ファナシス……それが、僕の名前だ」

<b>

<b>

<b>

――――――――――

<b>

<b>

<b>

「ハル・トレイター、見つけましたわ！」

<b>

食堂にクラウディアの鋭い声が響いた。

俺は気にすることなく、親子丼を食べる。

<b>

「……ハル、呼ばれているわよ？」

「……無視して、よろしいのですか？」

<b>

一緒にごはんを食べているアリスとアンジュが、ひそひそと言う。

<b>

「いや……本当はダメなんだろうけど、でもなんか、返事をしたらすごく面倒なことになりそうだから」

「確かに。ここ最近、ずっとハルをつけまわしているわよね」

「どちらの実力が上なのかハッキリさせておきたい……そのようなことを言っていますが、ハルさんは、なにか恨みを買うようなことを？」

「うーん……心当たりはないんだよね。それなりにうまくやっていけそうな雰囲気だったんだけど、でも、なんでこんなことになったのか」

「仲良くなったからこそ、ではありませんか？　男の子は、好きな女の子に対していじわるになると聞きます。その逆パターンという可能性は？」

「なるほどね。ハルが気になるから、ちょっかいをかけずにはいられない……うん、その可能性はあるかも。なんといっても、ハルだもの」

「ハルさんですからね」

<b>

二人は妙なところで意気投合していた。

俺だから、という理由がものすごく気になる。

<b>

「なにをわけのわからないことを言っていますの？　わたくしはただ、純粋に、編入生であるハル・トレイターの実力を、生徒会長であるわたくしが見極めてさしあげようと……」

「やあ、クラウディア。ようやく見つけたよ」

<b>

突然、第三者の声が乱入した。
